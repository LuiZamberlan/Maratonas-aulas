Dicas de Produtividade

Com o emmet abbreviation nós conseguimos completar partes do código muito mais rapidamente

Segurando o alt + shift e seta para cima ou para baixo, é possível duplicar a linha: teste teste

É também utilizando o 'emmet', ao colocar um * na frente da palavra a ser completada com o número de vezes que quer colocar o elemento e apertando o enter em seguida, será criada o número de cópias da linha
Ao querer escrever a mesma coisa em mais de uma linha, pode segurar alt e clicar nas outras linhas que deseja escrever que será criado o cursos de escrita para todas as linhas especificadas ou criar da mesma forma novos cursores nas linhas acima ou abaixo segurando o alt + ctrl
Segurar alt no início de uma linha, permite move-la para cima ou para baixo por meio das setas
Mesmo dando ctrl + z é possível errar o número de vezes que queria retroceder, para voltar para uma ação ja executada, mas que foi retrocedida, basta clicar ctrl + y.
